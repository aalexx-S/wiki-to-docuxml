Wiki to docuXML

Description

This project is aimed to create docuXML from a wiki page. Given wiki page URLs, we can generate a docuXML file using this project.

All the testing and developement are based on Chinese wiki pages (zh.wikipedia.org) for now.

Usage

  1. Upload or create a docuXML file.
  2. Edit URL list within various corpus names.
  3. Click update button to add wiki documents in docuXML.
  4. When finishing update,you can Click download to get the file.
